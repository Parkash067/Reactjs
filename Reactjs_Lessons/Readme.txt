What is React?
A javascript library used for user interfaces.
React allows you to build website or webpages.If any part of you webpage changes dynamically you can use react for

Uses of React
for notifications: We dont need to refresh page for every time .React provides us update and notifications of when update in realtime

Chat application: We dont need to update page for every time to see our messages it shows automatically react is handy for these kinds of work

React is only for the view it doesn't do any thing with database or server
or backend stuff. This only makes awesome views or design that updates 
the database or realtime

Components?
